ПРЕДИСЛОВЕ

Для того чтобы пользоваться этой программой, Вам понадобится Java Runtime Environment(JRE) версии не ниже 1.5 (это около 15 Мб, если что :)))) Установочный пакет JRE можно получить бесплатно на сайте компании SUN
http://java.sun.com/j2se/1.5.0/download.jsp

УСТАНОВКА И ЗАПУСК

Установка-1:
Распаковать в C:\Akira
Запускать ярлычком C:\Akira\Akira

Установка-2 (ну это несколько посложнее :))):
Распаковать куда угодно.
Заходите в ту папку, куда распаковали и видите файл akiraproject.jar Обычно, босле установки JRE архивы .jar будут автоматом открываться интерпретатором. Если винда попытается открыть чем-то ещё, выберите "Открыть с помощью", укажите что открыть надо Java(TM) 2 Platform Standard Edition binary  

Установка-3:
Распаковываете куда угодно.
Правите пути в ярлычке. Запускаете.

Постепенно будет инсталлятор - не забывайте, это даже пока не бета-версия :)
                      
После установки проверьте, чтобы папка, куда Вы распаковали архив с программой, не была защищена от записи.
